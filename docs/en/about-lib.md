# About the Library

### Description

[Network Forge](https://github.com/generaloss/network-forge) is a
lightweight and flexible Java library for building network applications and games.

### Requirements

**Java** 11+

### Dependencies

- *[Resource Flow](https://github.com/generaloss/resource-flow)*
  Used for convenient work with resources and data.
