# О библиотеке

### Описание

[Network Forge](https://github.com/generaloss/network-forge) - 
легкая и гибкая Java-библиотека для сетевых приложений и игр.

### Требования

**Java** 11+

### Зависимости

- *[Resource Flow](https://github.com/generaloss/resource-flow)* - 
используется для удобной работы с ресурсами и данными.
