# Пайплайн

## Что такое пайплайн

[EventPipeline](/src/main/java/generaloss/networkforge/tcp/pipeline/EventPipeline.java) -
это цепочка обработчиков
([EventHandlerLayer](/src/main/java/generaloss/networkforge/tcp/pipeline/EventHandlerLayer.java)),
через которую проходит каждое событие соединения:
- подключение
- отключение
- входящие данные
- исходящие данные
- ошибки

Это не просто список слушателей.
Это последовательная обработка, где каждый слой может:
- остановить дальшейшую обработку
- инициировать оправку
- изменить данные
- закрыть соединение
